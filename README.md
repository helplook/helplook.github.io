## ✨ 核心功能

### 1. AI 智能问答机器人
- 基于 ChatGPT/文心一言/豆包等大模型
- 支持多轮对话与上下文理解
- 自动捕捉关键词，7×24 小时在线服务
- 可嵌入网站、APP、小程序等多端

### 2. 智能语义搜索
- 超越传统关键词匹配，理解用户真实意图
- 支持口语化提问，精准定位知识内容
- 搜索结果高亮显示，提升查阅效率

### 3. 零代码内容管理
- 富文本编辑器 + Markdown 支持
- 自动保存与版本控制
- 批量导入导出（Word/PDF/Markdown）
- 多媒体内容嵌入（视频、图表、代码块）

### 4. 企业级权限管控
- 公开/密码保护/授权登录/混合访问模式
- 细粒度权限分配（栏目级、文章级）
- JWT 令牌安全验证，保障敏感信息安全

### 5. 全球化部署支持
- 20+ 语言一键自动翻译
- 自定义品牌域名与界面风格
- 适配企业 VI 体系，助力出海业务

### 6. 数据驱动运营
- 实时监测搜索热词、访客量、点击率
- AI 自动生成 SEO 关键词与描述
- 用户反馈收集与内容优化建议
